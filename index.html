Project Games API

Project Games API is a free REST API intended for web development students who want to connect to an API for coding projects.

The database provides various resources for game creation, including a deck of playing cards, a deck of tarot cards, coins, dice, and sets of words.

Project Games API is free and does not require an account to use.

Get started by reviewing the documentation and checking out some basic examples that were created with the API.

Documentation

The documentation provides every available GET endpoint. Each endpoint demonstrates the JSON response and description of all returned data.

Examples

The examples use the JavaScript Fetch API. Each example connects to one of the API's endpoints to use the JSON response to display resources from the database.
